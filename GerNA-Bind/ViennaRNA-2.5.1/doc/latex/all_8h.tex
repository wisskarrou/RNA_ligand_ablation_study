\doxysubsection{Detailed Description}
Energy evaluation for MFE and partition function calculations. 

,

This file contains functions for the calculation of the free energy $\Delta G$ of a hairpin-\/ \mbox{[} \mbox{\hyperlink{group__eval__loops__hp_gadf943ee9a45b7f4cee9192c06210dace}{E\+\_\+\+Hairpin()}} \mbox{]} or interior-\/loop \mbox{[} \mbox{\hyperlink{group__eval__deprecated_gaafbc187b7f78e8e82afb77dd6f3b8fc5}{E\+\_\+\+Int\+Loop()}}\mbox{]} .~\newline
 The unit of the free energy returned is $10^{-2} * \mathrm{kcal}/\mathrm{mol}$ 

In case of computing the partition function, this file also supplies functions which return the Boltzmann weights $e^{-\Delta G/kT} $ for a hairpin-\/ \mbox{[} \mbox{\hyperlink{group__eval__loops__hp_ga51fb555974f180b78d76142b2894851c}{exp\+\_\+\+E\+\_\+\+Hairpin()}} \mbox{]} or interior-\/loop \mbox{[} \mbox{\hyperlink{group__eval__deprecated_ga95de54d8a2a17645a95e0f34e189d9c9}{exp\+\_\+\+E\+\_\+\+Int\+Loop()}} \mbox{]}.
